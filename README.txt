# TRProject
Automating TradeRev's Job Portal as a part of coding challenge
---------------------------------------------------------------------------------------------------------------------
To clone and execute the tests in this project, please follow below commands. 
Also make sure the system has maven installed to run these tests and is a Windows OS system with chrome version 92
If not the chromedriver won't work and it will result in errors.
If the errors still persist please try with updating your local chrome browser
and/or installing the chromedriver according to your current chrome browser version in the 'Browserdrivers' folder.

```
git clone 
cd TRProject
mvn clean test
```

Observe the test results in the console for the specified coding challenge problems. 
You may also go to extent.html in the project folder and view the overall results/logs.
